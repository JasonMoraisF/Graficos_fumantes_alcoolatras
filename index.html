Sobre a base de dados

Este conjunto de dados é coletado do Serviço Nacional de Seguro Saúde da Coreia. O objetivo deste conjunto de dados é análise do sinal corporal e classificação do fumante ou bebedor.

O problema envolve a previsão de comportamentos de fumo e consumo de álcool com base em dados de sinais corporais.

Os dados podem abranger uma ampla gama de informações, como níveis de colesterol, triglicerídeos, hemoglobina, além de aspectos fisiológicos como a acuidade visual e auditiva.

O objetivo é desenvolver um modelo de inteligência artificial capaz de analisar esses dados e prever se uma pessoa é fumante, bebedora ou ambos.

Isso pode ter aplicabilidade em várias áreas, desde saúde pública até desenvolvimento de políticas de saúde e intervenções preventivas personalizadas. O desafio reside na coleta e análise eficazes desses dados de sinais corporais, bem como na construção de modelos precisos de previsão com base neles.
